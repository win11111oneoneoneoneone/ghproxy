GitHub File Accelerator

Release and archive use CF acceleration, files will redirect to JsDelivr

Note: project folders are not supported

Examples of legal input:

Branch source code: https://github.com/Username/Warehouse_name/archive/refs/heads/master.zip

Release source code: https://github.com/Username/Warehouse_name/archive/refs/tags/v0.1.0.tar.gz

Release file: https://github.com/Username/Warehouse_name/releases/download/v0.1.0/example.zip

Branch file: https://github.com/Username/Warehouse_name/blob/master/filename
